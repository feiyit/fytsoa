# 术语表（建议先看 3 分钟）

| 术语 | 说明 |
|---|---|
| 资产台账 | 资产主数据（编号、名称、分类、位置、责任人等） |
| 业务单据 | 资产发生业务动作的凭证（入库/领用/归还/调拨/处置等） |
| 盘点 | 核对系统记录与现场实物的一致性 |
| 维修工单 | 报修/维修过程记录与费用沉淀 |
| 保养计划 | 预防性维护：按周期触发保养工单/提醒 |
| 折旧配置 | 定义某资产怎么折旧（方法、折旧期、残值率等） |
| 折旧计提 | 按月生成折旧批次与明细，形成本期折旧金额 |
| 提醒规则 | “何时提醒谁”的模板 |
| 提醒任务 | 具体的一条提醒待办（规则的实例） |
| 留痕/溯源 | 记录资产相关关键操作的时间线，便于审计/追责/定位问题 |
| 仓库/库位 | 偏“库存管理”的存放位置（仓库更粗，库位更细） |
| 地点 | 偏“实际使用位置/安装位置”的地点树 |
| 责任人 | 对资产管理负责的人（资产管理员/保管责任） |
| 使用人 | 实际使用资产的人 |
| 期间 | 会计期间（例如 2026-01），折旧计提常用 |
