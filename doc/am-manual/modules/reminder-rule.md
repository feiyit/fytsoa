# 提醒规则（定义“何时提醒谁”）

提醒规则用于把“资产管理中的到期/应办事项”配置成可复用的规则，再由规则生成提醒任务推送给接收人。

## 1) 你在这里要做什么

- 选择规则类型（RuleType），例如：借用到期、质保到期、盘点到期、保养到期、调拨签收
- 配置提前天数（DaysBefore）：比如到期前 7 天提醒
- 配置接收人、渠道（站内/邮件/短信等，视系统支持）
- 启用/停用规则

## 2) 常见规则类型（RuleType）

- BORROW_DUE：借用到期
- WARRANTY_EXPIRE：质保到期
- INVENTORY_DUE：盘点到期
- MAINTENANCE_DUE：保养到期
- TRANSFER_SIGN：调拨签收

## 3) 配置方式建议

对普通用户：
- 推荐使用“可视化配置”（选择接收人、渠道等）

对系统管理员/高级用户：
- 可使用 JSON 配置（更灵活，但需要一定技术理解）

## 4) 与其它模块的关系

- 提醒任务：规则是模板，任务是实例
- 资产台账/单据/盘点/维修：任务的业务来源（BizType + BizId）
